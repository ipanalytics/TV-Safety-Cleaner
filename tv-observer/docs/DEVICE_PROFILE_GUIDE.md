# Device Profile Guide

Collect a fresh verified snapshot after every firmware update. A Controller profile must identify
platform, exact model, firmware/build, fingerprint, security patch assumptions, package categories,
health checks, and recovery evidence. Derive package claims from multiple observations and vendor
evidence; network labels alone are never enough.

Unknown platform, mismatched model/build/fingerprint, incomplete recovery readiness, and firmware
drift must refuse planning. TCL system profiles remain unverified templates until reviewed against a
real device snapshot and manual recovery drill. Never generalize a Fire TV Stick profile to a TCL TV.
