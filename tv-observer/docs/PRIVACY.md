# Privacy

Snapshots, observations, reports, and archives remain on the owner-controlled Raspberry
Pi. No external upload or analytics integration exists. Passwords, auth tokens, cookies, Wi-Fi
credentials, account secrets, and user message contents are excluded by default. Identifiers that
remain useful for compatibility can be removed with `redact` before sharing.

A user may ask local Codex to inspect a redacted copy, but Observer itself contains no AI/LLM client,
API credential, cloud storage, or automated external transfer.
